# Rocket League Stats API (Unreal Engine 5 Plugin)
Unofficial plugin for Unreal Engine 5 to easily work with data received from the Rocket League Stats API

## Installation
Download the latest version from the releases page and make sure that you put the plugin in the plugins folder of your Unreal Engine Project. 

Make sure Rocket League is configured for the Stats API to send payloads: https://www.rocketleague.com/developer/stats-api

## Usage
Set up your blueprint as the following screenshot to get the Rocket League data. The data string you can plug in Game Data or Player Data Converter to sort through the incoming data and use in your project.
